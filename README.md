# 徐老板盯盘

一个基于 Streamlit 的股票盯盘工具，支持实时行情监控、均线系统、成交量分析。

## 功能特性

-✅ 实时行情监控（自动刷新）
-✅ 均线系统（MA5、MA13、MA20）
-✅ 成交量监控（量比、换手率）
-✅ 自选股票管理
-✅ 量价关系分析
-✅ 深色主题界面

##快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
cd 股票监控
streamlit run app.py
```

### 3. 打开浏览器

自动打开 http://localhost:8501

## 使用说明

### 添加自选股票
在左侧边栏输入股票代码（如：600519）
- 点击"添加"按钮

### 刷新数据
- 手动刷新：点击"🔄 刷新数据"按钮
- 自动刷新：勾选"自动刷新"，设置刷新间隔

### 数据说明
- **实时价格**：当前最新价
- **MA均线**：5日、13日、20日移动平均线
- **量比**：当日成交量/30日平均成交量
- **换手率**：当日换手率

### 量价关系判断
- ✅ 放量上涨：价格>MA5 且 量比>1.5
- ⚠️ 放量下跌：价格<MA5 且 量比>1.5
- 📉 缩量：量比<0.5

## 数据来源

- 数据源：AKShare（免费、无需注册）
- 更新频率：实时行情约3秒延迟

## 注意事项

1. AKShare免费版有访问限制，请勿过于频繁刷新
2. 交易时间内数据实时更新，非交易时间显示最后收盘数据
3. 建议刷新间隔设置在15秒以上
